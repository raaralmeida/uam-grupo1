Análises

Esta seção apresenta os resultados preliminares das análises realizadas no âmbito do estudo sobre o potencial de implantação de sistemas de Urban Air Mobility (UAM).

Análise Espacial

A partir da base de dados de solicitações de voo de drones foi possível realizar uma análise espacial da distribuição das operações no território brasileiro. Essa análise permite identificar padrões de concentração que podem indicar áreas com maior potencial de demanda para serviços de mobilidade aérea urbana.

Identificação de Regiões Potenciais

Com base nos dados processados, serão avaliadas regiões metropolitanas que apresentam características favoráveis ao desenvolvimento de operações de UAM, incluindo fatores como densidade populacional, atividade econômica e conectividade urbana.

Localização de Vertiportos

Uma etapa futura do estudo envolve a identificação de possíveis locais para implantação de vertiportos, infraestrutura essencial para operações de aeronaves eVTOL. Essa análise considerará fatores como acessibilidade urbana, disponibilidade de espaço e integração com outros modos de transporte.

Em desenvolvimento

Esta seção será expandida com mapas, gráficos e resultados detalhados das análises espaciais e estimativas de demanda realizadas ao longo do projeto.
